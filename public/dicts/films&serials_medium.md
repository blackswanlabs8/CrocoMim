# Фильмы и сериалы — Medium

| term | description | about |
|---|---|---|
| Оператор | Специалист, управляющий камерой во время съёмок фильма. | Держать воображаемую камеру, крутить фокус, снимать. |
| Сценарист | Автор текста и диалогов для кино или сериала. | Писать в блокноте, зачеркнуть строку, показать «идея!». |
| Гримёр | Тот, кто наносит макияж актёрам перед съёмкой. | Наносить кистью грим, показать «старик/монстр!». |
| Костюмер | Специалист по подбору одежды для персонажей. | Примерять разные наряды, показать гардероб. |
| Продюсер | Человек, финансирующий и организующий создание фильма. | Считать деньги, подписывать контракт, кивать «да». |
| Монтаж | Процесс склеивания отснятых кадров в единый фильм. | Резать ножницами, соединять ленты, перематывать. |
| Дубляж | Озвучивание фильма на другом языке актёрами. | Говорить, приложив руку к губам, показать микрофон. |
| Закадровый голос | Речь за кадром, комментирующая или рассказывающая. | Указать за кадр, говорить без движения губ. |
| Титры | Список имён создателей в начале или конце. | Читать снизу вверх, показать «кто снимал». |
| Премьера | Первый показ фильма для публики и звёзд. | Пройти по красной дорожке, махнуть фанатам. |
| Оскар | Самая престижная награда в мировом кинематографе. | Поднять статуэтку, поклониться, улыбнуться счастью. |
| Кинофестиваль | Мероприятие, где показывают и оценивают новые фильмы. | Показать экран, жюри, аплодировать. |
| Бюджет | Сумма денег, выделенная на создание картины. | Считать пачку денег, показать «сколько можно тратить». |
| Кассовые сборы | Доход от проданных билетов в прокате. | Собирать билеты, считать кассу, улыбнуться. |
| Сиквел | Продолжение успешного фильма с теми же героями. | Показать «часть 2», продолжить историю жестами. |
| Приквел | История, рассказывающая события до оригинального фильма. | Показать «до этого», вернуться назад во времени. |
| Римейк | Новый фильм, переснимающий старую известную историю. | Показать старый кадр → новый, «обновили!». |
| Спин-офф | Отдельная история о второстепенном персонаже. | Указать на героя, вывести его вперёд. |
| Флэшбэк | Вставка из прошлого в текущую сцену. | Повернуть воображаемые часы назад, показать воспоминание. |
| Война | Крупный вооружённый конфликт между странами или армиями. | Стрелять, бежать в укрытие, показать солдат. |
| Шпионаж | Сбор секретной информации под прикрытием и обманом. | Шептать, передавать конверт, оглядываться. |
| Интрига | Скрытая игра интересов и тайные планы. | Показать «тайна», приложить палец к губам. |
| Предательство | Когда близкий человек нарушает доверие и помогает врагу. | Указать на друга → врага, показать «ты?!». |
| Месть | Ответное действие за причинённую обиду или боль. | Сжать кулаки, показать «я вернусь!». |
| Жертва | Человек, пострадавший от преступления или катастрофы. | Упасть, прижать руку к груди, стонать. |
| Свидетель | Тот, кто видел преступление и может рассказать. | Указать пальцем, показать «я видел всё!». |
| Тайна | Нечто скрытое, что герои пытаются раскрыть. | Прикрыть рот, показать «не говори никому». |
| Проклятие | Зловещее предсказание, приносящее несчастья героям. | Показать древний свиток, дрожать от страха. |
| Пророчество | Предсказание будущего, часто решающее судьбу мира. | Посмотреть в хрустальный шар, прошептать. |
| Портал | Магический проход между мирами или измерениями. | Провести круг в воздухе, шагнуть внутрь. |
| Параллельный мир | Альтернативная реальность с другими законами и героями. | Показать «зеркальный» мир, другой вариант себя. |
| Временная петля | Ситуация, когда герой снова и снова переживает один день. | Показать часы → повтор → часы → повтор. |
| Путешествие во времени | Перемещение героя в прошлое или будущее. | Крутить воображаемые шестерёнки, исчезнуть и появиться. |
| Искусственный интеллект | Разумная программа, способная думать и чувствовать. | Показать робота с «мозгом», говорить без эмоций. |
| Киборг | Существо, сочетающее живые ткани и механические части. | Показать металлическую руку, глаз-лазер. |
| Мутант | Человек с изменённой ДНК и необычными способностями. | Показать «странное» тело, использовать силу. |
| Суперсила | Необычайная способность, недоступная обычным людям. | Поднять тяжёлое, летать, стрелять лучами. |
| Лаборатория | Помещение для научных экспериментов и открытий. | Надеть очки, мешать колбы, «эврика!». |
| Эксперимент | Научная проверка гипотезы, часто с риском. | Надеть защиту, смешать жидкости, взрыв (условный). |
| Вирус | Микроскопическая угроза, вызывающая эпидемию или мутирование. | Кашлять, показать микроб, упасть. |
| Апокалипсис | Глобальная катастрофа, уничтожающая цивилизацию. | Показать руины, пепел, одинокого выжившего. |
| Выживание | Борьба за жизнь в условиях катастрофы или изоляции. | Строить укрытие, искать еду, дрожать от холода. |
| Колония | Поселение людей в чужом мире или на другой планете. | Построить домики, показать флаг на новой земле. |
| База | Укреплённое место для операций или укрытия. | Показать охрану, карту, «только свои!». |
| Подземелье | Тёмное подземное помещение с ловушками и сокровищами. | Спускаться по лестнице, зажечь факел, дрожать. |
| Крипта | Подземное захоронение, часто с проклятиями и призраками. | Открыть тяжёлую дверь, показать саркофаг. |
| Маяк | Башня с огнём, предупреждающая корабли об опасности. | Показать огонь на башне, волны внизу. |
| Ферма | Сельское хозяйство с животными, полями и домом. | Доить корову, кормить кур, косить сено. |
| Деревня | Небольшое поселение с домиками и колодцем. | Показать избушки, колодец, телегу. |
| Пустыня | Бескрайняя засушливая местность с песком и зноем. | Прикрыть глаза от солнца, идти по песку. |
| Джунгли | Густой тропический лес с дикой природой. | Расчищать лианы, показать обезьяну, змею. |
| Вулкан | Гора, извергающая лаву и пепел. | Показать извержение, бежать от лавы. |
| Антарктида | Ледяной континент на южном полюсе Земли. | Дрожать от холода, показать пингвинов, лёд. |
| Подводная лодка | Судно, способное плавать под водой незаметно. | Погрузиться, показать перископ, «тихо!». |
| Космический корабль | Транспорт для путешествий между планетами и звёздами. | Нажать кнопки, лететь, показать иллюминатор. |
| Спутник | Искусственный объект, вращающийся вокруг планеты. | Показать орбиту, кружить над Землёй. |
| Инопланетянин | Существо с другой планеты, часто с большими глазами. | Показать зелёную кожу, антенны, НЛО. |
| НЛО | Летающий объект неизвестного происхождения в небе. | Показать тарелку, световые лучи, испуг. |
| Голография | Объёмное изображение, возникающее в воздухе. | Провести рукой сквозь изображение, показать «не реально». |
| Киберпространство | Виртуальный цифровой мир, доступный через компьютер. | Надеть шлем, «войти в сеть», жесты как в «Матрице». |
| Хакер | Специалист, взламывающий компьютерные системы изнутри. | Печатать быстро, показать «взломал!», ухмыльнуться. |
| Детективное агентство | Офис, где частные сыщики расследуют дела. | Показать лупу, доску с нитками, «дело №1». |

| Двойной агент | Персонаж, предающий обе стороны, работая на всех. | Показать две маски, шептать разным людям. |
| Шифр | Секретный код для передачи закрытой информации. | Писать символы, разгадывать, показать «ключ!». |
| Артефакт | Древний предмет с магической или исторической силой. | Держать с осторожностью, светится, «не трогай!». |
